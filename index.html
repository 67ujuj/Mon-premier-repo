Cet exercice consistait à apprendre à utiliser la plateforme d'hébèrgement GitHub.

Nous avons créé un Repository depuis la plateforme GitHub, puis nous y avons exporté notre fichier grâce à Git.
Nous avons ensuite appris les commandes de base telles que "git init", "git add -A", "git commit -m", "git push origin master", et "git pull origin master".
